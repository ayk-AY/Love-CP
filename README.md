# 好きCP布教シートメーカー

好きなカップリングを布教するための、女性向け・同人向けの画像シート作成Webアプリです。

## 機能

- CP情報を複数登録
- 攻め・受けキャラ画像のローカルアップロード
- キャラごとのテーマカラー設定
- 5種類のデザインテーマ切り替え
- 完成プレビューのみをPNG保存
- localStorageとIndexedDBによるブラウザ内自動保存
- JSONエクスポート / インポート

## 起動方法

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開いてください。

## ビルド方法

```bash
npm run build
```

成果物は `dist/` に生成されます。

## GitHub Pagesへの公開方法

このプロジェクトはViteの `base: "./"` を設定しているため、GitHub Pagesのサブパス公開に対応しやすい構成です。

1. GitHubにリポジトリを作成して、このコードをpushします。
2. `npm run build` を実行します。
3. GitHub Actionsまたは任意のPagesデプロイ手段で `dist/` を公開します。

GitHub Actionsで公開する場合は、`dist/` をPages artifactとしてアップロードする設定にしてください。

## データ保存について

ログイン機能やサーバー保存はありません。

- 文字情報・テーマ設定は `localStorage` に保存されます。
- アップロード画像は `IndexedDB` に保存されます。
- 入力内容や画像はサーバーに送信されません。
- ブラウザのデータ削除、端末変更、別ブラウザ利用時にはデータは引き継がれません。
- データを引き継ぎたい場合はJSONエクスポートを使ってください。画像も可能な範囲でJSONに含めます。

## 注意

画像を公開する場合は、自作画像・利用許可のある画像・各作品や権利者のガイドラインに従ってください。
公式画像や他人のイラストの無断利用には注意してください。
