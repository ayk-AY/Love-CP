# 好きCP布教シートメーカー

好きなCPや関係性を布教するための、同人・推し活向け画像シート作成Webアプリです。

公開ページはこちらです。

https://ayk-ay.github.io/Love-CP/

## できること

- 複数のCP・関係性を登録
- 作品名やCP名、キャッチコピー、キャラ情報、コメントを入力
- 関係性に合わせて攻め・受け、または左側・右側のキャラ画像をアップロード
- 画像の表示位置や拡大率を調整
- `×`、`→`、`←`、`⇄`、`+` から関係性の記号を選択
- カップリングの場合は「リバ」表記を追加
- デザインテーマを切り替え
- 完成シートをPNG画像として保存
- SNS投稿向けに4件ずつ分割保存
- JSONバックアップでデータを引き継ぎ

## データ保存について

ログイン機能やサーバー保存はありません。

- 入力した文字情報や設定は、使用中のブラウザ内に保存されます。
- アップロード画像もブラウザ内で処理され、サーバーには送信されません。
- ブラウザのデータ削除、端末変更、別ブラウザ利用時にはデータは引き継がれません。
- データを引き継ぎたい場合は、アプリ内のJSON出力・JSON読込を使ってください。

## 注意

画像を公開する場合は、自作画像・利用許可のある画像・各作品や権利者のガイドラインに従ってください。

公式画像や他人のイラストの無断利用には注意してください。
